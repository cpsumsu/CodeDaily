> Problem: [53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/description/)

[TOC]

# 思路
> 貪心/dp

# 解题方法
> 取最大數字和，和小於0就重置數字

# 复杂度
- 时间复杂度:
> $O(n)$

- 空间复杂度:
> $O(1)$
  


# Code
```C++ []

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int s = 0, ans = nums[0];
        for (auto a : nums)
        {
            if (s > 0)
                s += a;
            else
                s = a;
            ans = max(ans, s);
        }
        return ans;
    }
};
```
