> Problem: [2085. 统计出现过一次的公共字符串](https://leetcode.cn/problems/count-common-words-with-one-occurrence/description/)

# 思路

> 兩個哈希表

# 复杂度

时间复杂度:
> $O(n)$

空间复杂度:
> $O(n)$



# Code
```C++ []
class Solution {
public:
    int countWords(vector<string>& words1, vector<string>& words2) {
        unordered_map<string,int> um1, um2;
        int ans = 0;
        for (auto a : words1)
            um1[a]++;
        for (auto b : words2)
            um2[b]++;
        for (auto [u,v] : um1)
            if (v == 1 && um2[u] == 1) ans++;
        return ans;
    }
};
```
