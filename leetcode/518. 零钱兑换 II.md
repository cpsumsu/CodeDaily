> Problem: [518. 零钱兑换 II](https://leetcode.cn/problems/coin-change-ii/description/)

# 思路

> 完全背包問題

# 复杂度

时间复杂度:
> $O(n * amount)$

空间复杂度:
> $O(n * amount)$

# Code
```C++
class Solution {
public:
    int change(int amount, vector<int>& coins) {
        int n = coins.size();
        vector<vector<int>> dp(n + 1, vector<int>(amount + 114));
        dp[0][0] = 1;
        for (int i = 0; i < n; ++i)
            for (int c = 0; c <= amount; ++c)
            {
                if (c < coins[i]) 
                    dp[i + 1][c] = dp[i][c];
                else
                    dp[i + 1][c] = dp[i][c] + dp[i + 1][c - coins[i]];
            }
        return dp[n][amount];
    }
};
```
