
> Problem: [162. 寻找峰值](https://leetcode.cn/problems/find-peak-element/description/)

[TOC]

# 思路

> 二分: 只要存在一個元素比任意兩邊數值大，那元素的其中一邊有峰值

- mid 和 mid + 1
- 如果 mid 大: 峰值在左邊
- 如果 mid 小: 峰值在右邊

# 复杂度

时间复杂度:
> $O(log(n))$

空间复杂度:
> $O(1)$

# Code
```C++ []
class Solution {
public:
    int findPeakElement(vector<int>& nums) {
        int l = 0, r = nums.size() - 1;
        while(l < r)
        {
            int mid = l + (r - l) / 2;
            if (nums[mid] > nums[mid + 1])
            {
                r = mid;
            }
            else
            {
                l = mid + 1;
            }
        }
        return l;
    }
};
```
