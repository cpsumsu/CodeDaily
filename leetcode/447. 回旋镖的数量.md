> Problem: [447. 回旋镖的数量](https://leetcode.cn/problems/number-of-boomerangs/description/)

# 思路

> 用哈希表記錄相同的回旋镖數量

# 复杂度

时间复杂度:
> $O(n)$

空间复杂度:
> $O(n)$



# Code
```C++ []
class Solution {
public:
    int numberOfBoomerangs(vector<vector<int>>& points) {
        int ans = 0;
        for (auto& pt1 : points)
        {
            unordered_map<int,int> dis;
            for (auto& pt2 : points)
            {
                int d2 = ((pt1[0] - pt2[0]) * 
                (pt1[0] - pt2[0])) +
                ((pt1[1] - pt2[1]) *
                (pt1[1] - pt2[1]));
                ans += dis[d2]++ * 2;
            }
        }
        return ans;
    }
};
```
