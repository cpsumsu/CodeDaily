> Problem: [2859. 计算 K 置位下标对应元素的和](https://leetcode.cn/problems/sum-of-values-at-indices-with-k-set-bits/description/)

# 思路

> 暴力枚舉，光速打卡下班

# 复杂度

时间复杂度:
> $O(n^2)$

空间复杂度:
> $O(1)$

# Code
```C++ []
class Solution {
public:
    int b(int n)
    {
        int cnt = 0;
        while(n)
        {
            if (n % 2 == 1) cnt++;
            n /= 2;
        }
        return cnt;
    }
    int sumIndicesWithKSetBits(vector<int>& nums, int k) {
        int ans = 0;
        for (int i = 0; i < nums.size(); ++i)
        {
            if (b(i) == k) ans += nums[i];
        }
        return ans;
    }
};
```
